Ola

Meu nome é Felipe de Oliveira, sou desenvolvedor web front-end junior, Violinista e técnico em programação de jogos digitais.

Formado na Etec de Ferraz de Vasconcelos, fui idealizador, programador e diretor na área de programação no desenvolvimento do meu TCC,

cujo o objetivo foi criar um jogo do zero.

Habilidades

JavaScript

Umas das minhas habilidades mais importantes e especiais.

Eventos, regristros, manipulação, validações, condições, dinamica e uma ótima estrutura de controle

com uma logica avançada, consigo evoluir e elevar a perfomance e complexidade para outro nivel de meus websites com esta ferramenta

CSS

Estilização, responsividade, atração, tudo isso possui um denominador comum

sim, o CSS, CSS é a ferramenta que eu gosto bastante de decicar bastante tempo

a frase, "a riqueza está nos detalhes", CSS consegue ser a definição desta frase

uma ferramenta poderosa que eu utilizo bastante para deixar os meus projetos responsivos e bem estilizados

HTML

E o HTML não poderia ficar de fora, a linguagem de hiper texto

foi umas das ferramentas que mais exige uma semantica boa e eficiente e um SEO e acessibilidade adequedos
